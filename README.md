# Experiment-2-GRAPHIC-PRIMITIVES
Aim

To create an Android application that draws basic graphic primitives such as line, rectangle, circle, oval and text using Canvas.

Algorithm

1. Open Android Studio.


2. Create a new Android project.


3. Create a custom class that extends View.


4. Declare a Paint object to set color and style.


5. Override the onDraw() method.


6. Use Canvas object to draw shapes.


7. Draw a line using drawLine().


8. Draw a rectangle using drawRect().


9. Draw a circle using drawCircle().


10. Draw an oval using drawOval().


11. Display text using drawText().


12. In MainActivity, create an object of the custom view.


13. Use setContentView() to display the view.


14. Run the application and observe the shapes drawn on the screen.
